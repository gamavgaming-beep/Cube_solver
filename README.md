# Rubik Solver Pro

A professional 3×3 Rubik's Cube Solver built with HTML, CSS and JavaScript.

- 🎨 Professional UI
- 📱 Mobile Responsive
- 🧩 3×3 Cube Editor
- 🎯 Color Picker
- ✅ Cube Validator

- 🎬 Step-by-Step Animation
- 🌙 Dark Theme
- ⚡ Fast Performance
- 🌐 GitHub Pages Ready
